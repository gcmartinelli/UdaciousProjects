Where did this come from?

I've been learning how to program by taking part in some Udacity courses. My first experience was with Udacity's CS101 course. When I built this site (May/2012) I was finishing Udacity's CS253. This is just some practice of what I've been learning.

Why did I make this?

I wanted to know what other Udacity students where building, so I set up this website as a reference point.

What did I use to build it?

This website is basically made in Python and it is hosted on Google App Engine. I've also been inspired by The Toolbox and I used some of it's tools!

Is the source code available?

No... Well, not yet at least. I still want to make some improvements before releasing the code. You can check out my other projects at GitHub.

Any tips for people wanting to learn how to program?

Yes! Look for courses at Udacity, CodeCademy and Khan Academy. It's never been easier to learn how to program, I wish I had this opportunity sooner! Regarding what language I recommend beginners... without a doubt: Python! :)
